# Крестики-нолики (Tic-Tac-Toe)

Консольное приложение на языке C++, в котором пользователь играет в крестики-нолики против компьютера. Программа поддерживает сохранение и загрузку данных игрока, ведёт статистику побед и обрабатывает ошибки пользовательского ввода.

## Возможности

- Игра против компьютера
- Проверка победы и ничьей
- Отображение информации об игроке
- Подсчёт количества сыгранных игр
- Подсчёт процента побед
- Сохранение данных в файлы
- Загрузка сохранённых данных
- Обработка неправильного ввода

## Использованные технологии

- C++
- Классы и объекты (ООП)
- Функции
- Двумерные массивы
- Работа с файлами (`fstream`)
- Генерация случайных чисел (`rand`)
- Обработка ошибок ввода

## Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/ВАШ-ЛОГИН/tic-tac-toe.git
```

### 2. Открытие проекта

Откройте файл решения `.sln` в Visual Studio.

### 3. Запуск программы

Нажмите:

```text
Ctrl + F5
```

или кнопку **Запуск без отладки**.

## Пример работы программы

```text
----Меню----
1. Начать новую игру
2. Показать инфо игрока
3. Статистика
4. Сохранить данные
5. Загрузить данные
6. Выход
```

Пример хода игрока:

```text
Введите ряд (1-3): 2
Введите столбец (1-3): 3
```

После завершения игры программа выводит результат:

```text
Вы выиграли!
```

или

```text
Вы проиграли!
```

или

```text
Ничья!
```

## Сохранение данных

Программа сохраняет:

- имя игрока;
- количество побед;
- количество сыгранных игр;
- состояние игрового поля.

Используются файлы:

```text
savePlayer.txt
saveGame.txt
```
